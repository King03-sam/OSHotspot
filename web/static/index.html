About OSHotspot

OSHotspot

Automatic WiFi Hotspot Manager for Linux

Share your computer's WiFi Internet connection with any device, even when NetworkManager hotspot fails.

Security

Localhost Only
Binds to 127.0.0.1 — never exposed externally
Session Token
Random 64-char token per launch, never persisted
Auto-Shutdown
Server stops after 2 hours of inactivity
Server-Side Validation
All config changes validated before being applied
